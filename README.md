# app-filterViaQuickbundles
Filter an input tractogram to remove abberant streamlines using [dipy's](https://dipy.org/) [quickBundles](https://dipy.org/documentation/1.0.0./examples_built/segment_quickbundles/) functionality.  Best performed immediately after tractogram creation.
Forked and heavily modified from from [app-extract-bundles-quickbundles](https://github.com/brainlife/app-extract-bundles-quickbundles)
